LANGKAH 1

Gunakan Formula Dasar di Excel.

Mulai dari penjumlahan, pengurangan, perkalian, dan pembagian. Semua dimulai dengan tanda sama dengan (=).

Formula paling mudah

1
Tambah: =A1+B1Jumlahkan nilai A1 dan B1.
2
Kurang: =A1-B1Kurangkan B1 dari A1.
3
Kali: =A1*B1Kalikan nilai A1 dan B1.
4
Bagi: =A1/B1Bagi nilai A1 dengan B1.
Shortcut: = lalu ketik formula

LANGKAH 2

Gunakan Function untuk analisis lebih canggih.

SUM, AVERAGE, COUNT, dan function lainnya membuat analisis data jadi lebih powerful dan cepat.

Function paling sering dipakai

1
SUM()Menjumlah range data. =SUM(A1:A10)
2
AVERAGE()Hitung rata-rata. =AVERAGE(B1:B10)
3
COUNT()Hitung jumlah cell. =COUNT(C1:C10)
4
IF()Logika kondisi. =IF(A1>100,"Besar","Kecil")
Hindari cara manual

Hitung satu-satu bikin error dan membuang waktu.

Pakai function

Excel → Formulas → Function Library.

LANGKAH 3

Buat Dashboard dan Chart.

Visualisasi data dengan chart membuat laporan jadi lebih mudah dipahami dan terlihat profesional.

Insert → Charts

Pilih jenis chart yang sesuai data kamu. Bar, Line, Pie, atau Column chart semua bisa dibuat otomatis dari data.
