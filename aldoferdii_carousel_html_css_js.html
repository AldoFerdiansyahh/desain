FONDASI WEB

Apa itu HTML, CSS & JS?

Tiga bahasa pemrograman yang bekerja sama menciptakan website modern. Mari kita kenali masing-masing fungsinya.

🏗️ HTML

Struktur - Kerangka & konten halaman web. Seperti tulang dalam tubuh.

🎨 CSS

Desain - Warna, layout, font. Membuat website indah dan menarik.

⚙️ JS

Interaksi - Logika dan animasi. Membuat website responsif & fungsional.

Analogi Sederhana: HTML = tulang, CSS = kulit & makeup, JavaScript = otak & gerakan. Ketiganya sama penting!

SYNTAX DASAR

Syntax Dasar HTML, CSS & JS

Setiap bahasa punya syntax sendiri. Ini adalah blok bangunan pertama Anda.

Contoh Syntax Masing-Masing

1
HTML Tags<div>, <button>, <p> - elemen halaman
2
CSS Rules.class { color: red; } - styling elemen
3
JS Functionsfunction doSomething() {} - logika & interaksi
💡 Tips:

Syntax itu mudah. Yang penting adalah pahami konsepnya.

⚡ Practice:

Banyak latihan = terbiasa dengan syntax

Jangan panik dengan syntax. Semua programmer pernah mengalami error. Practice makes perfect!

BEST PRACTICES

Best Practices untuk Code yang Baik

Mulai dari awal dengan habit yang baik. Ini akan menghemat waktu & energy di masa depan.

📝 Clean Code

Variable names yang jelas, comments yang bermakna, indentation yang rapi

♻️ DRY Principle

Don't Repeat Yourself. Gunakan functions & variables untuk reusable code

🧪 Testing Habit

Test setiap fitur yang Anda buat. Debug lebih awal = lebih efisien daripada nanti

Good Habits = Good Code. Mulai dengan praktik yang baik sekarang, daripada refactor nanti.

Siap Belajar Web Development?
